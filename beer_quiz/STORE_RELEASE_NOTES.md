# ストア公開用の重要情報

## 年齢制限

### Apple App Store
- **レーティング**: 17+
- アルコール関連コンテンツのため必須

### Google Play Store
- **レーティング**: 18歳以上
- ストア説明文に必ず以下を含めること: **「お酒は20歳になってから」**

## キーワード戦略（App Store）

### 主要キーワード
- ビール検定
- ビール検定対策
- クイズ
- 雑学

### 推奨タイトル例
- 「ビール雑学クイズ - ビール検定対策アプリ」
- 「ビール検定対策 - 雑学クイズで楽しく学習」

### 説明文のポイント
- 「ビール検定」は指名検索として強い（2026年現在）
- 「クイズ」と「雑学」の組み合わせが効果的
- SEO的には「ビール検定対策」という言葉を含めるのが有利

## ATT（App Tracking Transparency）対応

### 承認率が高い文言（実装済み）
「次画面で『許可』をタップしていただくことで、お客様の興味に合わせたビールの新製品情報やお得な広告をお届けできるようになります」

- ユーザー側のメリットを強調
- 2026年現在、最も承認率（Opt-in）が高い傾向

1. Google AdMobコンソールでアプリを登録

3. iOS / Android の広告ユニットIDが本番値であることを確認
4. ストア申請情報（App Privacy, Privacy Policy URL）を更新

## ビルドコマンド

### Android
```bash
flutter build appbundle --release
# または
flutter build apk --release
```

`android/key.properties` が未設定の場合は release ビルドが失敗します。  
テンプレート: `android/key.properties.example`

### iOS
```bash
flutter build ios --release
# Xcodeで署名とアーカイブ
```

### Web
```bash
flutter build web --release
```

### macOS
```bash
flutter build macos --release
```

## 注意事項

1. **広告表示**: クイズプレイ画面でバナー広告を表示（ホーム/結果画面では非表示）
2. **プライバシーポリシー**: App Storeの審査には必須。別途作成すること
3. **スクリーンショット**: 各プラットフォームのガイドラインに従って作成
4. **アイコン**: ビールをイメージした魅力的なアイコンを作成
